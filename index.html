Remarkable.js (a prototype)

Realtime inline commenting for dynamic online content

Simple yet powerful

Remarkable.js is a simple inline commenting platform powered by Firebase that allows users to converse about your content in realtime.

Painless setup

Built with the Firebase API, Remarkable.js is a quick drop-in to add a rich layer of interaction without the hassle of another backend. Just set up a Firebase, and you're ready to go!

Free and open source

Like all good things in life, Remarkable.js is free.

Installation

Backend

Front-end

Just add the following code to the head of your page:

$('article section').remarkable({
    selector: 'p',
    baseRef: 'https://remarkable.firebaseio.com/',
    scope: 'user_photos',
    inline_links: true
})
